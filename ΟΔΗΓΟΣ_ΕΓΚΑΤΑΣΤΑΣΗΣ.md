## Τι κάνει αυτό το σύστημα

Κάθε πρωί στις **08:30**, αυτόματα:
1. Ελέγχει **17+ sites** (ΔΥΠΑ, ΕΣΠΑ, Τουρκία, EURES, job boards, κλπ)
2. Αναγνωρίζει ευκαιρίες που ταιριάζουν στο προφίλ σου με **keyword scoring**
3. Κρατά μόνο τα **ΝΕΑ** (δεν επαναλαμβάνει παλιά)
4. Δημιουργεί **Apple Note** με όλα τα αποτελέσματα οργανωμένα ανά κατηγορία
5. Στέλνει **Mac notification** με σύνοψη
6. Στέλνει **deadline reminders** (7 μέρες, 3 μέρες, 1 μέρα πριν)

## ⏰ Αλλαγή ώρας εκτέλεσης

Αν θέλεις διαφορετική ώρα (π.χ. 09:00):

```bash
# Άνοιξε το αρχείο ρυθμίσεων
nano ~/Library/LaunchAgents/com.thanasis.opportunityhunter.plist
```

Βρες αυτό:
```xml
<key>Hour</key>
<integer>8</integer>
<key>Minute</key>
<integer>30</integer>
```

Άλλαξε τους αριθμούς, αποθήκευσε (`Ctrl+X` → `Y` → `Enter`), και μετά:

```bash
launchctl unload ~/Library/LaunchAgents/com.thanasis.opportunityhunter.plist
launchctl load ~/Library/LaunchAgents/com.thanasis.opportunityhunter.plist
```

## ❓ Troubleshooting

| Πρόβλημα | Λύση |
|---|---|
| Δεν φαίνεται Note | Έλεγξε αν η Notes app επιτρέπει Automation: System Settings → Privacy & Security → Automation |
| Δεν εμφανίζεται notification | System Settings → Notifications → Terminal → Allow |
| Script δεν τρέχει αυτόματα | `launchctl list \| grep thanasis` — αν δεν εμφανίζεται, κάνε reload |
| SSL errors στο log | Φυσιολογικό για μερικά ελληνικά sites — το script τα παρακάμπτει |
| Πολλά άσχετα αποτελέσματα | Αύξησε το score threshold: βρες `if score == 0` και άλλαξε σε `if score < 5` |

## 💡 Pro Tips

- **Κάθε Δευτέρα** τρέξε το χειροκίνητα για να δεις αν υπάρχουν weekend ανακοινώσεις

- **Πριν κάθε deadline** το σύστημα θα σε ειδοποιεί αυτόματα 7, 3 και 1 μέρα πριν
